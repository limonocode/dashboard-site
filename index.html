FAQ

Is it really free?

Yes. The core — unlimited groups and cards, all file importers, search, cleanup, export — is free forever. A future Pro plan adds optional cloud sync, live connectors and AI features.

Where is my data stored?

On your device, in your browser's local storage. There is no account and nothing is sent anywhere. Optional sync is opt-in and end-to-end under your control. See the privacy policy.

How does social import work without APIs?

Every platform legally provides a “download your data” export. You download that file and drop it into Dashboard — parsing happens locally. No scraping, no ban risk, no passwords.

When can I install it?

The extension is finished and in review preparation for the Chrome Web Store. This page will get the install button the moment it's live. Yandex Browser installs from the same store; Edge and Safari follow.
